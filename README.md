vim-config
==========

This are my „.vim”-files

Install
-------

```bash
$ git clone https://github.com/thomasba/dotvim.git ~/.vim
$ vim -nE +"colorscheme default" +PluginInstall +qall
```

Update
------

```bash
	$ git pull
```

If there are new plugins open vim and run:

	:PluginInstall

Update plugins
--------------

Open vim and run:

	:PluginUpdate
