vim-config
==========

This are my „.vim”-files

Install
-------

	$ git clone https://github.com/thomasba/dotvim.git ~/.vim
	$ vim +PluginInstall +qall

Update
------

	$ git pull

If there are new plugins open vim and run:

	:PluginInstall

Update plugins
--------------

Open vim and run:

	:PluginUpdate
